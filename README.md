# SQL for Data Analysts: Zero to Mastery 📊

Welcome to the **SQL for Data Analysts** repository! This is a complete, structured
series designed to take you from absolute beginner to job-ready SQL practitioner.
It covers every core concept a Data Analyst, BI Analyst, or Analytics Engineer needs
to query, clean, transform, and analyze real-world data using SQL.

## 📝 Topics Covered

Each section builds on the previous one, moving from fundamentals to advanced,
interview-level SQL:

1. **Introduction to SQL & Databases**: Relational database concepts, SQL syntax, and setup.
2. **SELECT, WHERE, ORDER BY**: Filtering, sorting, and basic querying.
3. **Joins (Inner, Left, Right, Full, Self)**: Combining data across multiple tables.
4. **Aggregate Functions & GROUP BY / HAVING**: Summarizing and analyzing data.
5. **Subqueries & CTEs**: Nested queries and Common Table Expressions for readable logic.
6. **Window Functions**: RANK, ROW_NUMBER, LAG/LEAD, and running totals.
7. **Data Cleaning with SQL**: Handling NULLs, duplicates, and data type issues.
8. **Query Optimization & Indexing**: Writing efficient, production-ready queries.
9. **Real-World Case Studies**: End-to-end business analysis projects using SQL.
10. **SQL Interview Questions**: Curated questions asked by top companies.
